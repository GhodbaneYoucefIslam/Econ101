# Econ101

Econ101 is an AI-powered chatbot optimized for answering economics-related questions. Built using **Chainlit**, it leverages **Elasticsearch** for efficient retrieval-augmented generation (RAG) and integrates with an **Ollama-powered LLM** to provide insightful and accurate responses.

## Features
- **Natural Language Understanding:** Handles a user prompts using the llama2 open source LLM.
- **Retrieval-Augmented Generation (RAG):** Uses Elasticsearch to fetch relevant documents before generating responses.
- **Enchanced responses:** Through prompt engineering using the LangChain library
